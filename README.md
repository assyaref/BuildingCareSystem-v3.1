# 📖 About

**Building Care System (BCS)** adalah aplikasi Enterprise Building & Facility Management yang dikembangkan untuk mendukung aktivitas operasional **Radiant Group Duri**.

Sistem ini membantu pengelolaan:

* 🏢 Building Maintenance
* 🛠 Corrective Maintenance
* 📅 Preventive Maintenance
* 📦 Asset Management
* ⚡ Utility Monitoring
* 📈 Dashboard Analytics
* 👥 User Management
* 📧 Email Notification
* 📋 Work Order
* ✅ Approval Workflow
